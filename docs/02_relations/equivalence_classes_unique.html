Description: Prove that equivalence classes, if they exist, are unique

prove equivalence_classes_unique: *A*R*C*D(is_equivalence_classes(R, A, C) & is_equivalence_classes(R, A, D) -> equal_sets(C, D)){ given |A, R, C, D|; implies both_equiv_classes; equiv_classes_C, equiv_classes_D = both_equiv_classes; return definition_uniqueness_lemma[<S: is_equivalence_class(R, A, S)>](C, D)(expand(equiv_classes_C), expand(equiv_classes_D)); }
