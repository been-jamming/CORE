Description: Prove that if a natural sum exists then the sum must be a natural number
prove natural_sum_is_natural: *X*Y*Z(is_natural_sum(X, Y, Z) -> Z in NATURALS){ given |X, Y, Z|; implies sum_Z; extract expand(sum_Z): X_in_NATURALS, Y_in_NATURALS, tuple_exists; extract natural_sum(X, Y)(X_in_NATURALS, Y_in_NATURALS)|OTHER_Z|: OTHER_Z_in_NATURALS, other_sum_Z; equal_Z = natural_sum_unique(X, Y, Z, OTHER_Z)(sum_Z, other_sum_Z); return equality_consistency[<Q: Q in NATURALS>](Z, OTHER_Z)(equal_Z)(OTHER_Z_in_NATURALS); }
